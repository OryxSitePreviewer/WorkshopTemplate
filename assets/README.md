# MDS Autowork — image assets

Drop the workshop photos here with these exact filenames so the site picks them up automatically:

| Filename | Where it appears | Notes |
|----------|------------------|-------|
| `storefront.jpg` | Hero background + Location section | The shopfront photo (MDS Autowork signboard). Landscape or portrait both fine; it's used as a cover-fit background. |
| `pickup.jpg` *(optional)* | Free Home Pickup section background | Any car / pickup-related photo. Optional — a dark fallback shows if absent. |

**Tips**
- Keep each file under ~500 KB for fast loading (resize to ~1600px wide, JPEG quality ~80).
- The hero already references `assets/storefront.jpg` — as soon as the file exists here, the hero photo goes live. No code change needed.
- The `/v1/` snapshot has its own `v1/assets/` folder; copy the same images there if you want v1 to show them too.
